# Module 5: Funnels & Landing Pages

🎯 **Objective:** Learn to build high-converting funnels and landing pages using GHL's drag-and-drop builder.

---

## 5.1 Funnels vs. Landing Pages vs. Websites

| Asset | What it is | Best for |
|-------|-----------|----------|
| **Landing Page** | A single, focused page with one goal (capture, sell, book) | Ad campaigns, lead magnets |
| **Funnel** | A multi-step sequence of pages guiding a visitor toward conversion | Sales sequences, webinar registrations |
| **Website** | A full, multi-page site with navigation | Brand presence, SEO content |

> 💡 **Tip:** GHL can build all three. Start with funnels and landing pages for quick wins, then add a full website later.

## 5.3 Building a Funnel in GHL

### Step 1: Create the Funnel
1. Go to **Sites** → **Funnels** → **"+ New Funnel"**
2. Choose: **Start from Scratch** or **Use a Template**
3. Name your funnel (e.g., "Free Consultation Funnel")

### Step 2: Add Funnel Steps

A typical lead-generation funnel has these steps:

| Step | Page Type | Purpose |
|------|-----------|---------|
| 1 | **Opt-in Page** | Capture name + email/phone |
| 2 | **Thank You Page** | Confirm submission, set expectations |
| 3 | **Booking Page** (optional) | Schedule a call directly |

A sales funnel might look like:

| Step | Page Type | Purpose |
|------|-----------|---------|
| 1 | **Sales Page** | Present the offer in detail |
| 2 | **Order Form** | Collect payment info |
| 3 | **Upsell Page** | Offer an add-on product |
| 4 | **Confirmation** | Order confirmation + next steps |

### Step 3: Design Each Page
- Use the **drag-and-drop builder**
- Add elements: Text, Images, Video, Buttons, Forms, Countdown Timers
- Built-in **image editor** for quick adjustments
- **Mobile-responsive** by default — always preview on mobile

### Step 4: Connect to Automation
- Form submissions → **Workflow triggers** (send email, create opportunity, add tag)
- Button clicks → **Redirect to next funnel step** or external URL

## 5.4 The Drag-and-Drop Builder

### Available Elements

| Category | Elements |
|----------|---------|
| **Layout** | Sections, Rows, Columns |
| **Content** | Text, Headlines, Images, Video, Dividers |
| **Forms** | Form Builder, Survey, Custom Fields |
| **Actions** | Buttons, Countdown Timer, Popup |
| **Commerce** | Order Forms, Pricing Tables, Coupons |
| **Advanced** | HTML/CSS blocks, Code snippets, Chat widgets |

### Design Tips
1. **One CTA per page** — don't confuse visitors with multiple actions
2. **White space matters** — let the design breathe
3. **Fast loading** — compress images, limit video autoplay
4. **Contrast for CTAs** — buttons should visually pop off the page
5. **Mobile-first** — test on mobile before desktop

## 5.5 Forms & Data Capture

### Creating a Form
1. Drag the **Form** element onto your page
2. Add fields: Name, Email, Phone, Custom Fields, Dropdown, etc.
3. Set the **form action:**
   - Redirect to the next funnel step
   - Show a thank-you message
   - Open a calendar to book

### Connecting Forms to Workflows
When a form is submitted, GHL can automatically:
- ✅ Create or update a **Contact**
- ✅ Add **Tags** and assign **Source**
- ✅ Start a **Workflow** (email sequence, SMS, pipeline)
- ✅ Create an **Opportunity** in a pipeline
- ✅ Notify a **team member**

## 5.6 A/B Testing

GHL supports **split testing** to optimize conversions:

1. Create **two versions** of a funnel step (variant A and B)
2. GHL splits traffic 50/50 (or custom %)
3. Track metrics: **page views, form submissions, conversion rate**
4. After enough data → pick the winner

### What to A/B Test
- Headlines and subheadlines
- CTA button text and color
- Hero images or videos
- Page layout (long-form vs. short-form)
- Form fields (fewer fields = higher conversion, but less data)

## 5.7 SEO for Funnels & Pages

For every page, configure:
- **Page Title** — descriptive, keyword-rich
- **Meta Description** — compelling 150-character summary
- **URL Slug** — clean and readable (`/free-consultation` not `/page-12345`)
- **Open Graph** — image and description for social sharing
- **Favicon** — professional touch
- **Custom Domain** — use your own domain, not a GHL subdomain
